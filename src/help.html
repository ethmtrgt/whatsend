WhatSend: Contact Non-Contacts with Ease

Tired of the hassle of adding someone to your phone contacts just to send them a quick message on Whatsapp Web? With our browser add-on, WhatSend, you can now send WhatsApp messages to non-contacts hassle-free. No need to clutter up your phone book with entries you'll only use once.

Usage

Using WhatSend is a breeze. Just type in the recipient's phone number with country code. Then hit the "contact" button.
That's it! A WhatsApp Web page will be opened, you'll be able to send messages through the page, and you didn't have to clutter your contacts with unnecessary entries.
If you would like to use a default country code, click on "Advanced" option and type in a country code. Add-on will save your entry and the country code will be written into the box every time you click it.
If you prefer not to use a default country code, just leave the desired box blank.

Source Code

Source code of the WhatSend is completely open source and available on github.com/ethmtrgt/whatsend. You can view its code or you can contribute to the new features if you would like to.
